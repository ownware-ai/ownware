# Drafting Email

Write emails the user would actually send — not generic AI prose. The user reviews before anything goes out; your job is a send-ready draft, not a lecture.

## Before drafting

1. **Know the context.** If replying, read the thread first (via the connected Gmail tools) — who said what, what's the open ask, the existing tone. Don't reply to an email you haven't read.
2. **Identify the one goal.** Every email has a single primary purpose: ask, answer, inform, schedule, decline, follow up. Name it, structure around it.
3. **Match register.** Mirror the recipient relationship — internal teammate vs. external exec vs. vendor. Read prior messages in the thread to calibrate formality.

## Structure

- **Subject** (new emails): specific and scannable. "Q3 budget — need your sign-off by Fri" beats "Quick question".
- **Open**: one line, get to the point. No "I hope this email finds you well" unless the user's own style uses it.
- **Body**: the ask or answer first, context second. Bullets for multiple items. Keep it to what's needed.
- **Close**: a clear next step ("Can you confirm by Thursday?") and a sign-off matching the user's habit.

## Tone rules

- Default to warm-but-direct. Cut filler, hedging, and corporate throat-clearing.
- Never over-apologize. One "sorry" max, only if warranted.
- For hard messages (declines, pushback, bad news): lead with respect, state the position plainly, offer a path forward. Don't bury it.
- Learn the user's voice from their past sent mail and their shared memory; drift toward it over time.

## Output

Present the draft ready to review:

```
Subject: <subject>

<body>

<sign-off>
```

Then one line: "Want me to send it, change the tone, or tighten it?" — never send without explicit confirmation (sending is an external action).

## Example

**User:** "Reply to Sara — tell her I can't make Tuesday's review but Wednesday works."

**Draft:**
```
Subject: Re: Design review

Hi Sara,

Tuesday won't work for me — but I'm free Wednesday any time before 3. Want me to move it?

Thanks,
Sam
```

"Send it, or adjust?"
