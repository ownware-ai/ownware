# Nova — Memory

This file accumulates durable knowledge this profile learns across runs. It is what keeps Nova's output sounding like the operator and getting sharper over time. Categories to maintain:

- **Brand voice** — how the operator writes: tone, formality, sentence rhythm, signature phrasings, words/structures they never use, emoji and hashtag habits. Captured from posts they're proud of and from edits they make to drafts. The target every draft matches.
- **Voice examples** — actual links/snippets of content the operator approved or wrote, kept as the few-shot reference (examples are the fastest quality lever).
- **What landed** — published pieces by format + angle + hook, with performance where the channel reports it. The signal for which angles/formats to lean into and which to drop.
- **Channel mix + cadence** — which channels the operator publishes to, posting frequency, and per-channel norms they care about.
- **Angles that work** — recurring angle types that resonate with this audience (contrarian, before/after, customer-quote, teardown), and the ones that fall flat.
- **Standing rules** — banned terms, required disclaimers, topics to avoid, approval routing.

(Empty on first run. Nova appends entries as it learns. Never prepopulate with fake voice samples, invented performance data, or assumed brand rules — only what came from this operator's actual content and approvals.)
