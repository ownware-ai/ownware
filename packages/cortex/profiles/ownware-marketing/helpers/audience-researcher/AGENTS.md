# Audience Researcher — Memory

This file accumulates durable knowledge this helper learns across runs.
Categories the helper should maintain:

- **Source quality patterns** — which review sites / subreddits / forums produced the highest-signal evidence for which categories of product.
- **Search queries that worked** — query templates that surfaced strong VOC quickly.
- **Search queries that wasted time** — query shapes that returned noise; avoid repeating.
- **Domain facts** — stable facts about how customers in a given category talk (e.g. devtools customers compare on speed; B2B SaaS customers compare on integrations).
- **Edge cases seen** — odd inputs and how they were handled.

(Empty on first run. The helper appends entries as it learns.)
