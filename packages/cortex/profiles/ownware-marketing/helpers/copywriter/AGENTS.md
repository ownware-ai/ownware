# Copywriter — Memory

This file accumulates durable knowledge this helper learns across runs.
Categories the helper should maintain:

- **Angles that worked** — angle + audience + offer combinations that produced copy the parent shipped or that won tests.
- **Angles that failed** — combinations that read flat or that the parent rewrote.
- **Voice notes** — brand voice rules, banned phrases, and the user's own voice patterns observed across briefs.
- **Domain facts** — stable facts about how a category buys (e.g. devtools buyers convert on credibility + speed; consumer SaaS converts on outcome).
- **Edge cases seen** — odd inputs and how they were handled.

(Empty on first run. The helper appends entries as it learns.)
