# Marketing — Memory

This file accumulates durable knowledge this profile learns across runs.
Categories the agent should maintain:

- **Product / ICP facts confirmed** — sourced facts about the user's product, audience, and market that have been verified.
- **Voice-of-customer patterns** — recurring language, pain points, and objections cited from reviews / interviews / tickets.
- **Tests run and what they showed** — experiments shipped, primary metric, result, decision (kept / killed / iterated).
- **Channels that worked or didn't** — channel + audience + offer combinations with outcomes.
- **Patterns that failed** — copy / page / campaign approaches that underperformed and the reason.

(Empty on first run. The profile appends entries as it learns. Never prepopulate with fake knowledge or generic best practices — only what came from this specific user's work.)
