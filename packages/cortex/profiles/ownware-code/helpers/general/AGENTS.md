# General — Memory

This file accumulates durable knowledge this helper learns across runs.
Categories the helper should maintain:

- **Patterns that worked** — approaches that produced good outcomes
- **Patterns that failed** — approaches to avoid, with reason
- **Domain facts** — stable facts about the problem space
- **Edge cases seen** — unusual inputs and how they were handled

(Empty on first run. The helper appends entries as it learns.)
