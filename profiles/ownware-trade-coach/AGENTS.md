# Trader memory

This file accumulates what the Coach has learned about the specific
trader using this profile. Treat every entry as evidence, not opinion.

The Coach reads this file at the start of every session and uses it
to spot repeats — third time this quarter the trader has added to a
losing breakout, fifth time this month they've revenge-sized after
an open red bar, etc.

## Format

Entries are short, dated, and specific. No prose paragraphs. No
generic advice. Every entry names the trade, the rule, and the
dollar cost where known.

```
- 2026-04-30 — added to losing $NVDA position 12 min after first stop hit. Cost $740. Pattern: averaging down on momentum names.
```

## Sections

### Recurring rule breaks
*(Coach appends one line per repeat. After the same break appears
3+ times, the Coach calls it out by name in live sessions.)*

### Tilt triggers
*(What state precedes the trader's worst sequences. e.g. "after
3+ early-morning losses", "Mondays after losing weeks", "during
FOMC week".)*

### Winner pattern (evolving)
*(The actual setup that pays the trader, as derived by
`extract_shadow_strategy`. Updated after each shadow-account review.
This is the rule the trader already follows when winning, surfaced.)*

### Lessons that stuck
*(Observations the trader explicitly accepted. Anything here is
fair to quote back when the same situation reappears.)*
