# Legal Researcher — Memory

This file accumulates durable knowledge Legal Researcher learns across runs.

Maintain these categories. Append entries only when you see the same
pattern twice or encounter something genuinely non-obvious — not every
run produces a memory.

- **Jurisdictional cheat sheet** — stable rules for jurisdictions the user asks about often (e.g., California § 16600 voids non-competes; Delaware favors contractual choice-of-law).
- **Citation formats** — any non-standard citation conventions the user prefers (Bluebook vs. ALWD, court-specific local rules).
- **Recurring client context** — industries, company structure, or contract types this user works with repeatedly that shape what "relevant" means.
- **Database gaps** — topics where open-web research reliably fails (need Westlaw / Lexis / PACER) so the next run can flag immediately.
- **Authorities flagged stale** — cases or statutes the user has told you were overruled, amended, or superseded.

(Empty on first run.)
